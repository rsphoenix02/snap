# Monorepo Restructure + Codebase Cleanup

**Date:** 2026-03-22
**Status:** Approved

## Goal

Restructure SNAP from a single Next.js app (`app/`) into a monorepo layout (`frontend/`) with the git repo at the root, and remove all dead/redundant code.

## 1. Git Restructure

- Move `app/.git/` up to `snap/.git/` (promotes repo to monorepo root)
- Rename `app/` to `frontend/`
- Delete `frontend/.next/` and `frontend/node_modules/` (will be regenerated)
- Create a root-level `.gitignore` covering `node_modules/`, `.next/`, build artifacts, `.env*`
- Run `npm install` inside `frontend/` to regenerate `node_modules/`
